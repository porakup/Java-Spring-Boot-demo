# java-backend-test

This project for test purpose.

# Installation

At project directory terminal run this for build
```bash 
mvnw clean install
```
After build success then run this for start
```bash
mvnw spring-boot:run
```
Or run this for test
```bash
mvnw test
```

## Usage

open swagger-ui
```
http://localhost:8080/swagger-ui/
```
