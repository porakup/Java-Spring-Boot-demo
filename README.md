# java-backend-test

This project for test purpose.

# Installation

at project directory terminal run this for build
```bash 
mvnw clean install
```
after build success then run thie for start
```bash
mvnw spring-boot:run
```
run this for test
```bash
mvnw test
```

## Usage

open swagger-ui
```
http://localhost:8080/swagger-ui/
```
